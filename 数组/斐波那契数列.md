## 题目描述

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0，第1项是1）。
n<=39

***示例***

输入:4

返回值:3

```C++
class Solution {
public:
    int Fibonacci(int n) {
        int result[2]={0,1};
        if(n<2)
            return result[n];
        int fibN=0;
        int fibNMinusOne=0;
        int fibNMinusTwo=1;
        for(int i=2;i<=n;i++){
            fibN=fibNMinusOne+fibNMinusTwo;
            fibNMinusOne=fibNMinusTwo;
            fibNMinusTwo=fibN;
        }
        return fibN;
    }
};

```
